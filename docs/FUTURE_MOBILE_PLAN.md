# Gelecek Mobil Planı

Mobil uygulama bu sürümde geliştirilmedi. Mimari, API tabanlı istemcilere açık olacak şekilde tasarlandı.

## Yaklaşım

- Hasta portalı önce PWA olarak genişletilebilir.
- Daha sonra React Native istemci aynı auth ve hasta endpointlerini kullanabilir.
- Mobil istemcide sadece hasta rolüyle başlanması önerilir.

## Gereken Ekler

- Refresh token veya kısa ömürlü access token mimarisi
- Push notification
- Mobil PDF görüntüleme
- Randevu hatırlatma
- Biyometrik giriş opsiyonu
